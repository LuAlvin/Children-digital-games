# 数字大炮游戏

适合6岁儿童的数学教育游戏，通过有趣的游戏方式学习加减法。

## 游戏介绍

这是一个寓教于乐的数学游戏，小朋友通过发射炮弹击中降落伞上的怪物来回答数学题。游戏包含多个关卡，难度逐渐提升，帮助孩子循序渐进地掌握加减法运算。

## 游戏特色

- 适合6岁儿童设计
- 可爱的怪物表情和动画效果
- 三个难度级别，循序渐进
- 关卡升级系统
- 友好的用户界面
- 无干扰的游戏体验

## 关卡设计

### 第一关：11以内加减法
- 加法：两个数相加不超过11
- 减法：结果非负，最大被减数为10
- 升级要求：获得100分

### 第二关：20以内加减法
- 加法：两个数相加不超过20
- 减法：结果非负，最大被减数为19
- 升级要求：获得100分

### 第三关：100以内加减法
- 加法：两个数相加不超过100
- 减法：结果非负，最大被减数为99
- 通关要求：获得300分

## 游戏玩法

1. 观察降落伞上怪物携带的数学题
2. 在输入框中输入答案
3. 按下回车键或点击发射按钮发射炮弹
4. 答案正确：击中怪物，获得10分
5. 答案错误：怪物会提示"小朋友，答案不对哦"，继续降落
6. 怪物落地前未击中：失去一条生命
7. 生命耗尽：游戏结束

## 游戏机制

### 得分系统
- 每答对一题获得10分
- 每获得100分可以升级到下一关
- 选择"留在当前关卡"会清零分数，可以反复练习

### 生命系统
- 初始3条生命
- 怪物落地未击中失去1条生命
- 生命耗尽游戏结束

### 关卡升级
- 达到升级分数时弹出升级提示
- 可以选择：
  - 开始新关卡：进入下一关，难度提升
  - 留在当前关卡：分数清零，继续练习当前难度
- 每个关卡的升级提示可以重复触发

### 通关奖励
- 完成第三关（获得300分）后显示通关祝贺
- 可以选择重新开始或退出游戏

## 技术特点

### 气泡消息系统
- 答案错误时显示气泡消息而非弹窗
- 气泡消息2秒后自动消失
- 不打断游戏流程

### 游戏暂停/恢复
- 弹窗出现时自动暂停游戏
- 停止所有游戏循环（怪物生成、降落、倒计时）
- 用户选择后恢复游戏并生成新题目

### 响应式设计
- 适配不同屏幕尺寸
- 流畅的动画效果
- 友好的视觉反馈
