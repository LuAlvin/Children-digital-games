# 音频文件说明

本游戏需要以下音频文件来提供完整的游戏体验：

## 📁 音频文件列表

### 1. 背景音乐 (Background Music)
- **文件名**: `background.mp3` / `background.ogg`
- **用途**: 游戏进行中的背景音乐
- **格式**: MP3 和 OGG（兼容不同浏览器）
- **音量**: 0.3（适中音量）
- **循环**: 是
- **触发**: 游戏开始时播放，游戏结束时停止

### 2. 击中音效 (Hit Sound)
- **文件名**: `hit.mp3` / `hit.ogg`
- **用途**: 答案正确，击中怪物时播放
- **格式**: MP3 和 OGG
- **音量**: 1.0（正常音量）
- **触发**: 炮弹击中怪物时

### 3. 答案错误音效 (Error Sound)
- **文件名**: `error.mp3` / `error.ogg`
- **用途**: 答案错误时播放
- **格式**: MP3 和 OGG
- **音量**: 1.0（正常音量）
- **触发**: 输入错误答案时

### 4. 升级音效 (Level Up Sound)
- **文件名**: `levelup.mp3` / `levelup.ogg`
- **用途**: 关卡升级时播放
- **格式**: MP3 和 OGG
- **音量**: 1.0（正常音量）
- **触发**: 达到升级分数时

### 5. 通关音效 (Game Complete Sound)
- **文件名**: `complete.mp3` / `complete.ogg`
- **用途**: 游戏通关时播放
- **格式**: MP3 和 OGG
- **音量**: 1.0（正常音量）
- **触发**: 完成所有关卡时

### 6. 时间紧迫音效 (Urgent Sound)
- **文件名**: `urgent.mp3` / `urgent.ogg`
- **用途**: 倒计时少于5秒时播放
- **格式**: MP3 和 OGG
- **音量**: 1.0（正常音量）
- **触发**: 倒计时从5秒变为4秒时

### 7. 游戏失败音效 (Game Over Sound)

## 🎵 音频文件获取建议

### 免费音频资源网站
1. **Freesound.org** - 提供大量免费音效
2. **Zapsplat** - 游戏音效资源
3. **Bfxr.net** - 在线音效生成器
4. **OpenGameArt.org** - 游戏美术和音频资源

### 推荐音频类型
- **背景音乐**: 轻松愉快的儿童音乐
- **击中音效**: 爆炸或击中声效
- **错误音效**: 友好的错误提示音
- **升级音效**: 庆祝或升级音效
- **通关音效**: 胜利或完成音效
- **紧迫音效**: 警示或紧迫音效
- **失败音效**: 失败或游戏结束音效

## 📝 注意事项

1. 音频文件应放在 `audio/` 文件夹中
2. 建议同时提供 MP3 和 OGG 格式以确保兼容性
3. 音频时长不宜过长，建议：
   - 背景音乐：30-60秒循环
   - 音效：1-3秒
4. 音频文件大小建议控制在合理范围内，避免影响加载速度

## 🔧 技术实现

游戏已通过 `audioManager` 对象实现音频控制：
- 初始化音频元素
- 播放/停止控制
- 错误处理和日志记录
- 音量控制和重置功能
